2. Add Project → name it (e.g. "BTS-Tracker") → Create Project
3. Left sidebar → Realtime Database → Create Database → any location → Test mode → Enable
4. Copy the Database URL shown (e.g. https://bts-tracker-abc-default-rtdb.firebaseio.com)
5. ⚙️ gear → Project Settings → Web app </> → Register → copy the apiKey value
